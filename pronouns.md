## 人称代词

| mi,ri,lo | 单数           | 复数           |
|----------|--------------|--------------|
| 主格       | mi/or-,ri,lo | zi,di,sio    |
| 宾格       | min,rin,lon  | zin,din,sinn |
| 属格       | mo,ril,lom   | zo,dih,siom  |
| 与格       | ur,ruz,loz   | oth,duz,sos  |

特别的，以下词用于在第三人称双人语境下指代另外一个

|    | 单数   |
|----|------|
| 主格 | lodh |
| 宾格 | lonc |
| 属格 | lob  |
| 与格 | loq  |
