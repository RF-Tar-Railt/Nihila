## 拼合规则

以 `-(n)ia/-ná` 举例：
- 元音结尾时：
  - 若后缀的元音首字母形式与结尾元音相同或可合并，则使用元音首字母形式：
  nahi -> nahia
  - 否则使用辅音首字母形式：
  ádo -> ádoná
- 辅音结尾时：
  - 若单词的结尾辅音是 `r` 或 `l`, 则去除辅音，然后按照元音规则实行：
  manil -> mannia
- 若单词结尾是双辅音，且第一个是 `n`, 第二个是 `t` 或 `d` 或 `h` ,且后缀首字母为n时，则去除两个辅音，然后使用辅音首字母形式：
  - tent -> tená
- 否则使用元音首字母形式：
  ág -> ágia
