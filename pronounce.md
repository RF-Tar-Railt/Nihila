## 音变

### 连音

- a + e = á
- a + i = í
- e + a = é
- u + y = y
- y + u = iu
- e + i /é + i = ei

### 首辅音

鼻音变体(上一个词尾为n)：

- v -> b
- b -> m
- p -> f

塞音变体(上一个词尾为éd)：

- p -> b
- t -> z

软变体：

- m -> v
- b -> v
- c -> g
- t -> d
- r -> n

### 多音节

#### c1vc2vc3v结构：

c2后的v向下置换

如tamana -> tamena

derehe -> derohe

nisici -> nisyci/nisaci

#### c1v1c2v2c2v2结构:

第二个c2进行首辅音变换

如 tamama -> tamava -> tameva

docici -> docigi

#### c1v1c2v2c3v2结构、且c2c3相近:

v2 换成 v1

如fénala -> fénéla
